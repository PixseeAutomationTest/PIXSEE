# **PIXSEE AUTOMATION** 
## 啟動步驟
1. 把這個github連結到自己的電腦
2. 打開powershell
3. 輸入pip install -r requirements.txt
4. 確認檔案裏面都沒有毛毛蟲紅線
5. powershell輸入appium --use-plugins=inspector --allow-cors
6. 看pycharm那邊有沒有第一個test出現
